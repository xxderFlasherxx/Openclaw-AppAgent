COPILOT SYSTEM-PROMPT FORMAT
=============================

Wenn Hans (OpenClaw) eine Aufgabe an den VS Code Copilot sendet,
wird folgender Kommentar-Block an den Anfang der Zieldatei geschrieben:

TEMPLATE:
---------

/*
 * ═══════════════════════════════════════════════════════════════
 * AUFGABE FÜR DIESES SCRIPT:
 * {detailedPrompt}
 *
 * TECHNISCHE ANFORDERUNGEN:
 * - Unity 2022.3 LTS
 * - C# 9.0
 * - URP kompatibel
 * - Namespace: {gameName}
 *
 * ABHÄNGIGKEITEN:
 * {dependencyList}
 *
 * ERWARTETER CODE:
 * {expectedStructure}
 * ═══════════════════════════════════════════════════════════════
 */

using UnityEngine;

// TODO: Implementiere {taskDescription}


PLATZHALTER:
- {detailedPrompt}     → Der ausführliche Prompt von Günther
- {gameName}           → Projektname (PascalCase)
- {dependencyList}     → Liste der bereits existierenden Dateien
- {expectedStructure}  → Kurze Beschreibung der erwarteten Klassen/Methoden
- {taskDescription}    → Einzeiler-Zusammenfassung


ALTERNATIVE (für ganze Dateien die Hans direkt schreibt):

Hans kann auch direkt den Code schreiben den Günther vorgibt,
ohne den Copilot zu nutzen. In diesem Fall wird die Datei komplett
von Hans erstellt basierend auf dem detailedPrompt von Günther.

VS CODE CLI REFERENZ
====================

Grundlegende Befehle:
- code [ordner]              → Ordner in VS Code öffnen
- code [datei]               → Datei in VS Code öffnen
- code --goto [datei:zeile]  → Datei an bestimmter Zeile öffnen
- code --diff [a] [b]        → Diff zwischen zwei Dateien
- code --wait                → Wartet bis Fenster geschlossen wird
- code --new-window          → Neues Fenster öffnen
- code --reuse-window         → In bestehendem Fenster öffnen

Extension-Management:
- code --install-extension [id]    → Extension installieren
- code --uninstall-extension [id]  → Extension deinstallieren
- code --list-extensions           → Installierte Extensions auflisten

Wichtige Extensions für GameDev:
- github.copilot              → GitHub Copilot
- github.copilot-chat         → Copilot Chat
- ms-dotnettools.csdevkit     → C# Dev Kit
- visualstudiotoolsforunity.vstuc → Unity Tools (optional)

VS Code Tasks (tasks.json):
- Können über Terminal getriggert werden
- code --folder-uri [pfad] && tasks ausführen
- Nützlich für Build-Automatisierung

VS Code Settings (für Copilot):
{
  "github.copilot.enable": {
    "*": true,
    "csharp": true
  },
  "github.copilot.advanced": {},
  "editor.inlineSuggest.enabled": true
}

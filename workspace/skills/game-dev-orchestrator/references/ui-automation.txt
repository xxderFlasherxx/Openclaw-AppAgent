UI-AUTOMATION FÜR VS CODE (Linux)
==================================

Option 1: xdotool (empfohlen für Linux)
-----------------------------------------
Installation: sudo apt install xdotool

Befehle:
  xdotool search --name "Visual Studio Code" windowactivate
  xdotool key ctrl+shift+i          # Copilot Chat öffnen
  sleep 1
  xdotool type --delay 10 "[PROMPT]" # Text eintippen
  xdotool key Return                 # Enter drücken
  sleep 30                           # Warten auf Generierung
  xdotool key Return                 # Accept
  xdotool key ctrl+s                 # Speichern

Beispiel-Script:
  #!/bin/bash
  # VS Code aktivieren
  WID=$(xdotool search --name "Visual Studio Code" | head -1)
  xdotool windowactivate "$WID"
  sleep 0.5

  # Copilot Inline Chat öffnen (Ctrl+I)
  xdotool key ctrl+i
  sleep 1

  # Prompt einfügen via Clipboard (sicherer als tippen)
  echo -n "$PROMPT_TEXT" | xclip -selection clipboard
  xdotool key ctrl+v
  sleep 0.5
  xdotool key Return

  # Warten bis Copilot fertig ist
  sleep 30

  # Accept + Save
  xdotool key Return
  sleep 0.5
  xdotool key ctrl+s

Option 2: xdg-open + wmctrl
-----------------------------
  wmctrl -a "Visual Studio Code"   # Fenster fokussieren
  xdg-open [datei]                 # Datei öffnen

Option 3: xclip für Clipboard
-------------------------------
  echo "TEXT" | xclip -selection clipboard   # In Clipboard schreiben
  xdotool key ctrl+v                         # Einfügen

WICHTIG FÜR LINUX:
- X11 muss laufen (nicht Wayland), oder ydotool für Wayland nutzen
- Display-Variable muss gesetzt sein: export DISPLAY=:0
- In VirtualBox: Guest Additions müssen installiert sein
- Timing ist kritisch: zu schnell → Fehler, lieber großzügig warten
